#### 使用说明

```shell
hexo n --path about/me "标题"    # 创建 source/_posts/about/me.md
hexo g           # generate
hexo s           # server，本地预览
hexo d           # deploy
hexo clean       # 清缓存，诡异 bug 先跑这个
hexo g -d        # 生成 + 部署 二合一
```
